# Milestone V1 M1

## Historical Note

This milestone captured the shift from a graph-first homepage to the current
cover page plus dedicated `/network` explorer.

## Lasting Outcomes

- `/` became a cover page with stats and search
- `/network` became the dedicated full-network experience
- the header gained navigation and download actions
- the search bar gained reusable props for placement and initial value

## What Changed Since The Original Note

  screenshot artifacts; those have been removed because they were not portable.
- The structure-viewing workflow and layout-cache system were added later and
  are not part of this milestone itself.
